🖥️ Hardware

El hardware es la parte física del computador, es decir, todos los componentes que se pueden ver y tocar. Sin el hardware, el software no podría funcionar, ya que necesita un soporte físico para ejecutarse. El hardware incluye desde el procesador hasta el teclado y el mouse.

⚙️ Componentes Internos
Dentro del computador

Son las piezas que se encuentran dentro del gabinete y permiten el funcionamiento del sistema:

  • CPU (Procesador) — Es el cerebro del computador. Ejecuta las instrucciones y procesa la información.
  • RAM (Memoria) — Almacena temporalmente los datos que el procesador necesita en ese momento. A más RAM, mejor rendimiento.
  • Disco Duro (HDD/SSD) — Almacena permanentemente el sistema operativo, programas y archivos del usuario.
  • Placa Madre — Es la tarjeta principal que conecta todos los componentes del computador entre sí.
  • Tarjeta Gráfica (GPU) — Procesa y genera las imágenes que se muestran en el monitor.
  • Fuente de Poder — Convierte la corriente eléctrica y distribuye energía a todos los componentes.
⌨️ Dispositivos de Entrada
Input

Son los dispositivos que permiten al usuario ingresar datos o instrucciones al computador:

  • Teclado — Permite escribir texto y ejecutar comandos.
  • Mouse — Permite mover el cursor y seleccionar elementos en la pantalla.
  • Micrófono — Captura sonido para grabaciones o videollamadas.
  • Cámara web — Captura imágenes y video en tiempo real.
  • Escáner — Convierte documentos físicos en imágenes digitales.
🖨️ Dispositivos de Salida
Output

Son los dispositivos que muestran o entregan el resultado del procesamiento al usuario:

  • Monitor — Muestra la información visual generada por el computador.
  • Impresora — Imprime documentos e imágenes en papel.
  • Altavoces — Reproducen el sonido generado por el sistema.
  • Proyector — Proyecta la imagen del computador en una superficie grande.
💾 Dispositivos de Entrada y Salida
Input / Output

Son dispositivos que pueden tanto enviar como recibir datos:

  • Pantalla táctil — Funciona como monitor (salida) y como dispositivo de toque (entrada).
  • Disco externo — Permite guardar y leer archivos.
  • Memoria USB — Almacena y transfiere datos entre dispositivos.
  • Tarjeta de red — Envía y recibe datos a través de una red.
📦 Tipos de Hardware
Clasificación

El hardware se puede clasificar según su función dentro del sistema:

  • Hardware de procesamiento — CPU y GPU, encargados de procesar datos.
  • Hardware de almacenamiento — HDD, SSD, USB, almacenan información.
  • Hardware de comunicación — Tarjetas de red, módem, permiten conectarse a internet.
  • Hardware periférico — Todos los dispositivos externos como teclado, mouse, monitor.
