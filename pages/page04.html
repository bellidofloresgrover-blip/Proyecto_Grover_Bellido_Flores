🌐 Introducción a Redes

Una red informática es un conjunto de computadoras y dispositivos conectados entre sí para compartir recursos, información y servicios. Las redes permiten la comunicación entre personas, el acceso a internet, el intercambio de archivos y el uso compartido de dispositivos como impresoras. Sin las redes, cada computadora funcionaría de forma aislada.

📡 Tipos de Redes
Clasificación por tamaño

Las redes se clasifican según su tamaño y cobertura geográfica:

  • LAN (Local Area Network) — Red de área local. Cubre espacios pequeños como una casa, oficina o escuela. Es rápida y económica. Ejemplo: el Wi-Fi de tu casa.
  • MAN (Metropolitan Area Network) — Red de área metropolitana. Cubre una ciudad o campus universitario. Conecta varias redes LAN.
  • WAN (Wide Area Network) — Red de área amplia. Cubre países o continentes. El ejemplo más grande es Internet.
  • PAN (Personal Area Network) — Red de área personal. Cubre pocos metros. Ejemplo: conexión Bluetooth entre tu celular y audífonos.
🖧 Dispositivos de Red
Hardware de red

Son los equipos físicos que permiten crear y mantener una red:

  • Router — Dirige el tráfico de datos entre diferentes redes y reparte el internet a los dispositivos conectados.
  • Switch — Conecta múltiples dispositivos dentro de una misma red local y permite que se comuniquen entre sí.
  • Módem — Convierte la señal del proveedor de internet para que los dispositivos puedan utilizarla.
  • Access Point — Extiende la cobertura de la red de forma inalámbrica (Wi-Fi) a zonas donde no llega el cable.
  • Tarjeta de red (NIC) — Componente interno que permite a una computadora conectarse a la red por cable o inalámbricamente.
🔒 Protocolos de Red
Reglas de comunicación

Un protocolo es un conjunto de reglas que define cómo se comunican los dispositivos en una red. Sin protocolos, los equipos no podrían entenderse entre sí.

  • HTTP/HTTPS — Protocolo para la navegación web. HTTPS es la versión segura con cifrado.
  • IP — Protocolo de Internet. Asigna una dirección única a cada dispositivo en la red.
  • TCP — Garantiza que los datos lleguen completos y en el orden correcto a su destino.
  • DNS — Traduce nombres de dominio como google.com a direcciones IP numéricas.
  • FTP — Protocolo para transferir archivos entre computadoras en una red.
🔗 Topologías de Red
Formas de conexión

La topología es la forma en que están organizados y conectados los dispositivos en una red:

  • Estrella — Todos los dispositivos se conectan a un punto central (switch o router). Es la más usada por su facilidad de mantenimiento.
  • Bus — Todos los dispositivos comparten un mismo cable principal. Si el cable falla, toda la red se cae.
  • Anillo — Cada dispositivo se conecta al siguiente formando un círculo. Los datos viajan en una sola dirección.
  • Malla — Cada dispositivo se conecta con todos los demás. Es muy confiable pero costosa de implementar.
