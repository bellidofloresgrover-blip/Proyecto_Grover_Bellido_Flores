SOFTWARE

Parte Lógica del Computador

💾 ¿Qué es el Software?

El software es el conjunto de programas, instrucciones y datos que permiten a una computadora funcionar y realizar tareas específicas. Es la parte lógica del sistema informático, es decir, lo que no se puede tocar, pero que controla el hardware.

🖥️ Software de Sistema

Es el encargado de administrar y controlar el funcionamiento del hardware y permitir que otros programas se ejecuten.

  • Windows
  • Linux
  • Android
  • Drivers (controladores)
💻 Software de Programación

Son herramientas que utilizan los programadores para crear otros programas o aplicaciones.

  • Python
  • Java
  • C++
  • Visual Studio Code
📱 Software de Aplicación

Son programas diseñados para que el usuario realice tareas específicas.

  • WhatsApp
  • Google Chrome
  • Microsoft Word
  • Excel
⚙️ Software Utilitario

Son programas que ayudan a mantener y optimizar el funcionamiento del sistema.

  • Antivirus
  • CCleaner
  • WinRAR
  • Desfragmentador de disco
